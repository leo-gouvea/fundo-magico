# Fundo Mágico — Gerador de Fundos alimentado por IA
https://leo-gouvea.github.io/fundo-magico/

## 🧠 Sobre o Projeto

**Fundo Mágico** é uma aplicação web que gera **fundos em HTML e CSS** a partir de descrições em linguagem natural utilizando **Inteligência Artificial generativa**.
O projeto integra uma **interface front-end** com um **fluxo de automação no n8n**, que se comunica com o **Google Gemini** para gerar código front-end limpo e reutilizável de forma dinâmica.

---

## ✨ Funcionalidades

* Geração de fundos a partir de texto
* Código **HTML + CSS** gerado por IA
* Integração com **API REST** via webhook do n8n
* Interface simples e responsiva
* Código gerado limpo e reutilizável

---

## 🛠️ Tecnologias Utilizadas

* **HTML5**
* **CSS3**
* **JavaScript (ES6+)**
* **API REST**
* **n8n (automação de fluxos)**
* **Google Gemini (IA generativa)**

---

## 🔄 Como Funciona

1. O usuário descreve o fundo desejado
2. O front-end envia a requisição para um **webhook do n8n**
3. O n8n processa a solicitação e chama o **Gemini AI**
4. A IA gera o código HTML e CSS
5. O código gerado é retornado e exibido na interface

---

Este projeto foi desenvolvido para praticar:
